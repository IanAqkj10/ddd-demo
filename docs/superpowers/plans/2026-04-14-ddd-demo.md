# DDD Demo Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建一个可阅读、可测试的 DDD 教学型多模块项目，展示下单与取消订单两个核心用例。

**Architecture:** 采用 Maven 多模块单体结构，按 `domain / application / infrastructure / interfaces / start` 分层。核心业务规则放入聚合和领域服务，MyBatis-Plus 仅负责持久化实现。

**Tech Stack:** Java 8, Spring Boot 2.7, MyBatis-Plus, Maven, JUnit 5
